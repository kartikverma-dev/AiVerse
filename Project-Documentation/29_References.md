# References

## Why it exists
To list scientific papers, blog posts, and code repositories cited during system design.

## When it was introduced
Maintained since 2025.

## What problem it solved
It credits the researchers and frameworks that made the project possible.

## Alternative ideas discussed
- No citations. Dismissed because it undermines academic integrity.

## Why the final decision was chosen
A dedicated references file respects intellectual property and assists researchers.

## Future possibilities
- Automatic generation of BibTeX outputs for academic citations.
