# Patent Potential

## Why it exists
To examine patentable elements, like the semantic lineage graph generation pipeline.

## When it was introduced
Analyzed in 2026.

## What problem it solved
It provides intellectual property protection guidelines for the founders.

## Alternative ideas discussed
- Relying entirely on trade secrets. Dismissed because open-source components expose core logic.

## Why the final decision was chosen
Patenting the backend pipeline provides defensible IP while keeping the frontend client open-source.

## Future possibilities
- Filing a provisional patent for the automated status calculation algorithm.
