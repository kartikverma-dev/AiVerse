# Technology Stack

## Why it exists
To outline the selected stack: Next.js 15, Tailwind CSS v4, Supabase, Framer Motion, and NVIDIA NIM.

## When it was introduced
Selected in 2025.

## What problem it solved
It ensured high development velocity, stunning visuals, and modern React 19 features.

## Alternative ideas discussed
- Vite + React + Express. Dismissed because Next.js App Router provides built-in SSR, API routing, and better SEO.

## Why the final decision was chosen
The stack represents the bleeding edge of modern web development, matching the cutting-edge theme of the project.

## Future possibilities
- Upgrading to React Server Components where interactive states are not required to improve loading speeds.
