# Experimental Design

## Dataset
We compiled a seed database of 15 core AI concepts ranging from historical paradigms (Prompt Engineering) to bleeding-edge developments (Loop Engineering, Model Context Protocol).

## Test Protocol
1. Run the automatic status calculation pipeline weekly.
2. Log all computed status evaluations.
3. Compare computed states with manual evaluations provided by three senior AI architects.
4. Calculate precision, recall, and F1-scores.
