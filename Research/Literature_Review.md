# Literature Review

## Semantic Web and Ontologies
Ontological engineering (e.g., OWL, RDF schemas) has long attempted to catalog domain knowledge. However, manual curation by domain experts is a major bottleneck.

## Citation Graph Analysis
Traditional bibliometric analysis relies on static citation counts. While robust, citation counts lag actual industry adoption by 12 to 24 months.

## Large Language Models for Ontology Extraction
Recent advances in structured outputs (e.g., function calling, json schemas) enable LLMs to parse unstructured academic texts and developer documentation to extract semantic relations with high accuracy.
